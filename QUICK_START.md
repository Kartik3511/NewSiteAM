## Next Steps:

### 1. Add Your Images
Place your project images in `assets/images/` folder. The site uses placeholders that will be replaced with your actual images.

**Recommended image names and sizes:**
- `hero-bg.jpg` (1920x1080px) - Main hero background
- `work1.jpg` to `work5.jpg` (800x1000px) - Recent work
- Category preview images (various sizes - see README.md)

### 2. Update Content
Edit the HTML files to:
- Replace placeholder text with your actual project descriptions
- Add your real project titles and details
- Update service lists with your specific offerings
- Add your social media links in the footer

### 3. Customize Colors (Optional)
In `assets/css/style.css`, update the color variables:
```css
:root {
    --primary-bg: #ffffff;
    --accent-beige: #d4c5b0;
    /* Add your brand colors */
}
```

### 4. Test Responsiveness
The site is already responsive! Test it on:
- Desktop browsers
- Tablets
- Mobile phones

## How to View:

Simply open `index.html` in any modern web browser!

Or run a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using PHP
php -S localhost:8000

# Using Node.js (http-server)
npx http-server
```

Then visit: http://localhost:8000
